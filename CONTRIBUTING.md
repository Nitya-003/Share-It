# 🤝 Contributing to Share-IT

First off, thank you for considering contributing to Share-IT! It’s people like you that make this a great tool for everyone.

### How Can I Contribute?

1.  **Reporting Bugs:** Open an issue and describe the bug with steps to reproduce it.
2.  **Suggesting Enhancements:** Have an idea for a new feature? Open an issue to discuss it!
3.  **Pull Requests:** * Fork the repo.
    * Create a branch (`git checkout -b feature/AmazingFeature`).
    * Commit your changes (`git commit -m 'Add AmazingFeature'`).
    * Push to the branch (`git push origin feature/AmazingFeature`).
    * Open a Pull Request.

### Code Style
* Please use TypeScript for frontend components.
* Ensure your code follows the existing folder structure.
* Write clear, commented code for complex logic.
